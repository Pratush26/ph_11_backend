# InfraCare - Backend

This is a backend project of an mern stack project. It is a project about public Infastucture issue solver platform.

### Links
- [Live link 1](https://infracare-b79a4.web.app)
- [Live link 2](https://infracare-b79a4.firebaseapp.com)
- [Github - Frontend](https://github.com/Pratush26/ph_a11_frontend)
- [Github - Backend](https://github.com/Pratush26/ph_11_backend)

### Used Tech
- Express js
- MongoDB
- Firebase Admin SDK
- Stripe

### Key Features
- User can do CRUD operation with Issues
- Admin and staffs can update their status
- User can purchase features with stripe payment method
- User authenticaiton and authorization base oparations

### Installation Process
- Clone this repo in move to the parent file
- Attach the environment veriables
- Then run _
- `npm install`
- `npm run dev`
